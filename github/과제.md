git merge
    fast-forward
        브랜치가 직선으로 이어질때
        단순 커밋 포인터만 이동
        분기 없음
    3-way merge
        브랜치가 갈라졌다 다시 합쳐질 떄
        병합 커밋이 생성됨
        분기 있음

    merge conflict 충돌
    같은 파일의 같은 부분 다르게 수정되어있으면 충돌
    수동 수정 새로운 커밋

html
    HyperText Markup Language
    웹 페이지의 구조 정의
    제목 문단 이미지 링크 같은 웹 페이지의 뼈대 만듦

    '브라우저'에서 html을 해석해 화면 구성함

    태그들의 조합으로 구성
    태그는 <태그> 형태
        <여는태그>콘텐츠</닫는태그> 

        <h1~6>heading 제목</h1>
        <p>paragraph문단 내용</p>
        <br> break line줄바꿈
        <hr> horizontal line가로선
        태그는 중첩되며 계층구조
    
<!DOCTYPE html> 이 파일은 html5로 만들어짐을 선언함
<html lang="ko">
    <head>
        <!-- 안 보임, 메타데이터 -->
        <title>탭 제목</title>
    </head>
    <body>
        화면에 보임
    </body>
</html>
